## 📖 About

Study tool covering the complete CCNA 2 v7.0 Final Exam syllabus. Every question includes per-option explanations so you understand not just what is correct, but **why every other option is wrong**.

Topics covered:

- VLANs, Trunking & Inter-VLAN Routing
- Spanning Tree Protocol (STP, RSTP, PVST+)
- EtherChannel (LACP, PAgP)
- DHCPv4 & DHCPv6
- HSRP & First-Hop Redundancy
- Static & Dynamic Routing (OSPF)
- IPv4 & IPv6 Addressing
- NAT & PAT
- Wireless Networking (WLANs, WPA2, WLC)
- Switch Security (Port Security, DAI, DHCP Snooping, 802.1X)
- Network Troubleshooting
